Movie recommendation system



Mockup project: as a team of 4 we had to developp a webapp for a cinema owner. His business was falling behind and our goal was to create a movie recommendation system to suggest potential future popular movies.

We started to clean then analyze the database using Python, Pandas and Plotly then I built the app with a Python library named Streamlit. Based on the massive IMDb database I trained a KNN Machine Learning model to find movies with similar genres to the one selected. The Streamlit App is available here.
