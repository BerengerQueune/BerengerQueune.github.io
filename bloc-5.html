Industrialization of a machine learning algorithm



Link to Github repository
Tech stack: Python, Streamlit, Docker, MLflow, Heroku, Plotly, , Sklearn (Linear Regression), Fast API

Get Around is a company helping people to rent their car. They want a tool to help them find if it would be beneficial to set a minimum time delta between two car rentals and another tool using Machine Learning to find the rental price of a car. I used Streamlit to create a web application to find if setting a time delta would help grow the company's revenue. Here is my Streamlit app.
Then I made a Linear Regression and saved it in MLflow to predict the rental price of a car with Fast API.
