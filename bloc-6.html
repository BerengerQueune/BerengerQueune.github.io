API - Data Visualisation online



Link to Github repository
Tech stack: Python, Streamlit, Docker, AWS Elastic Beanstalk, Fast F1, Plotly, Matplotlib

This was a group project. Our goal was to gather data related to Formula One from an API to propose various visualizations to fans. I focused my work on the deployment phase. I made our web application with Streamlit and Docker to deploy it on Heroku for testing then on AWS Elastic Beanstalk for our presentation.
