Predictive analysis of Walmart weekly sales with Machine Learning



Link to Github repository
Tech stack: Python, Plotly, Sklearn, GridSearchCV

From a list of the 35 most touristic French cities, I Made API requests to gather their weather data and GPS coordinate. Once done, I scrapped data from Booking.com to find the 20 most popular hotels for each cities. I then saved all these information in a CSV file and stored it in a Amazon S3 Datalake. After that, I have extracted and modified the Data to save it in a PostgreSQL Redshift Datawarehouse. Finally, I made SQL queries to request the data and create a visualisation on a map.
